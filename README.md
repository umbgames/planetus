# 🌍 Planet:Us
### Explore, Build, Take

**Planet:Us** is an **infinite planetary exploration strategy game** where players discover, expand, and compete across a procedurally generated universe.

Built with **TypeScript**, the game focuses on scalable systems, procedural generation, and multiplayer-style interaction across a shared galaxy.

The universe never ends — every planet discovered adds to the ever-growing frontier.

---

## 🚀 Overview

Planet:Us blends **exploration, empire building, and MMO-style gameplay** into a constantly expanding universe.

Players travel through space discovering new planets, establishing colonies, gathering resources, and expanding their influence. Strategy and exploration go hand-in-hand as players compete or cooperate within a living galaxy.

---

## ✨ Core Features

### 🌌 Infinite Universe
Procedurally generated planets ensure the universe never runs out of places to explore.

### 🧭 Exploration
Discover new planetary systems, environments, and hidden opportunities across space.

### 🏗 Strategic Expansion
Build colonies, manage resources, and expand your interplanetary network.

### 🤝 MMO-Inspired Gameplay
Interact with other explorers across the galaxy through trade, alliances, and competition.

### ⚙️ TypeScript-Powered Architecture
The entire project is written in **TypeScript**, enabling:

- strong typing
- scalable systems
- maintainable game logic
- modern development workflows

---

## 📱 Cross-Platform

Planet:Us is designed to run on multiple platforms from the same codebase.

Supported platforms include:

- 🖥 **PC**
- 📱 **Mobile**
- 🌐 **Web Browsers**

This allows players to explore the universe anywhere.
